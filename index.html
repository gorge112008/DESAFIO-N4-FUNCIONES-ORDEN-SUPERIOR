SIMULADOR INTERACTIVO DE VENTA DE COMIDA RÁPIDA

PRIMERA ENTREGA DEL PROYECTO FINAL

*Descripción del Proyecto:

El proyecto se basa en la creación de un Simulador Interactivo de Venta de comida rápida.

  • En esta primera versión el simulador contará con las siguientes funcionalidades:

    =>Selección de menús variados para distintas carteras.
    =>Elaboración de pedidos detallados.
    =>Cancelación de pedidos elaborados.
    =>Mostrar la carta detallada del conjunto de menú seleccionado.
    =>Mostrar boleta detallada del pedido y tiempo de espera total.
    =>Opción de salida del simulador sin hacer ningún pedido.

Base de la pre-entrega: Algoritmo de simulación de un negocio de venta de comida rápida con la creación del menú de comida como objetos con distintas propiedades mediante una función constructora, agregando los menús dentro de un array y aplicando un filtro para obtener nuevos arrays con variados conjuntos de menús.
Definiendo métodos y funciones se ejecutará el correcto pedido o cancelación de cada menú e imprimirá un reporte detallado de cada menú.
Antes de finalizar la simulación si el pedido se hizo correctamente se emitirá una boleta de venta detallada y el tiempo de espera, caso contrario solo se emitirá un mensaje por alerta y se finalizará la simulación.
